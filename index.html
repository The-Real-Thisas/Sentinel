Privacy

Sentinel is built from the ground up to value your privacy above all else. When using Sentinel you are anonymous, all traffic to and from Sentinel is encrypted and travels through the tor network. All software included with sentinel is open-source and thoroughly checked to insure no data leakage can occur.

Security

Sentinel is one of the most secure operating systems out of the box. This is possible because sentinel is pre-configured to be as secure as possible. Core steps are taken to insure this, such as; Closing all unused ports, Strict firewall rules, system-wide application sandboxing and our Sentinel Anti-Virus.

Open-Source Code

You don't have to trust us, you can check out the code for yourself. We believe to be truly private and secure, the code should be open-source and the community should be able to criticize and test for issues and vulnerabilities. That way, you don't have to take our word for it.
